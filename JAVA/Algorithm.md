## 1. 两变量交换
  1. 中间变量法，缺点 int 相加超出范围  
  ```java
    int a = 3;
    int b = 5;
    int temp = a;
    a = b;
    b = a;
  ```
  2. 相加法
  ```java
    a = a + b;
    b = a - b;
    a = a - b;
  ```
  3. 使用异或,缺点: 逻辑不清晰
   ```java
    a =a ^ b;  // a = 3 ^ 5;
    b = a ^ b; // b = 3 ^ 5 ^ 5 = 3
    a = a ^ b; // a = 3 ^ 5 ^ 3 = 5
   ```
